# CodeExplain - Plain-English Code Tutor

A polished, frontend-only Streamlit shell for an AI SaaS product that explains code in plain English.

## What is included

- Custom top navigation with theme toggle and placeholder actions
- Premium hero section
- Large syntax-highlighted code editor with line numbers
- Placeholder process indicator
- Tabbed results layout for explanation, complexity, line-by-line breakdown, improvements, and quiz
- Slim right-side information rail
- Footer and modular component structure

## Tech stack

- Python
- Streamlit
- streamlit-ace

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Notes

- No backend logic is implemented.
- No LLM calls or API calls are included.
- The app is intentionally designed as a launch-ready UI shell for later integration.
